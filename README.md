# student-portfolio
personal-portfolio
Personal Portfolio Website

This project is a clean, modern, and responsive personal portfolio website built using HTML, CSS, and JavaScript. It is designed to present a student profile in a professional way, suitable for internship submissions, resumes, and academic presentations. The website includes multiple pages, clear navigation, and well-structured content.

Features

Multi-Page Structure

Home

About Me

Projects

Contact

Simple and clear navigation bar on all pages.

Professional Layout

Blue and white theme for a formal appearance.

Clean typography and spacing.

Easy-to-read sections and structured alignment.

Responsive Design

Works on mobile, tablet, and desktop screens.

Layout adjusts automatically based on screen size.

Detailed Sections

About page includes skills, education, and profile summary.

Projects page includes descriptions of real completed projects.

Contact page includes email and social links.

Lightweight and Fast

No frameworks.

Built only with HTML, CSS, and JavaScript.

Loads quickly on all devices.

Technologies Used

HTML for structure

CSS for design and layout

JavaScript for interactivity and navigation handling

Project Structure

index.html
about.html
projects.html
contact.html
style.css
script.js
assets/ (optional images)

How to Run

Download or clone the repository.

Open the file index.html in any browser.

The entire website will work without installation.
