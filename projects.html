Projects — Detailed

A selection of my most significant projects with technical details and responsibilities

Student Portfolio — Activity Points Calculation

Built a web-based Student Portfolio system to evaluate and calculate activity points from uploaded certificates. The system reduces manual verification by using NLP to extract text and classify certificate types.

Key features

  • Certificate upload & OCR/NLP extraction
  • Rule-based & ML-assisted category mapping
  • Student & teacher dashboards with filters and timelines
  • Secure REST APIs and MySQL storage

Tech: ReactJS • Node.js • Express • MySQL • NLP libraries

NeuroPrompt — AI-Driven Cognitive Learning Platform (In Progress)

NeuroPrompt is an adaptive learning platform that personalizes content to a student's cognitive profile: visual, auditory, reading preference, and attention patterns. It uses quizzes and interaction metrics to tailor content.

Key features

  • Cognitive assessment via quizzes and interaction telemetry
  • AI-driven lesson generation & lesson variants (text/audio/visual)
  • Adaptive dashboard showing strengths, suggestions & progress
  • Chat-style UX for on-demand topic explanations

Tech: ReactJS • Node.js • MongoDB • Adaptive Engine
